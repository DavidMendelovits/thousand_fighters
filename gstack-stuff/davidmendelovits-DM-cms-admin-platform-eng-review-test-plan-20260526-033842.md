## Affected Pages/Routes
- POST /new/pipeline/run — SSE streaming pipeline orchestration endpoint
- GET /new/pipeline/{id} — Pipeline run status endpoint

## Key Interactions to Verify
- POST with characterId + brief → SSE events for each step in order
- SSE events include step name, status (started/progress/complete/error), and artifacts
- Steps parameter filters which pipeline steps run
- onProgress callback pipes Codex stdout through SSE events

## Edge Cases
- Invalid characterId (empty, special chars)
- Unknown step name in steps array
- Pipeline step failure mid-run (e.g., image gen timeout)
- Client disconnects during SSE stream
- GET /new/pipeline/{unknownId} returns 404
- Concurrent pipeline runs for different characters

## Critical Paths
- Full pipeline: concept → draft → sprites → normalize → QA → publish
- Selective pipeline: steps=['normalize','qa'] with pre-existing assets
- Legacy routes remain functional after /new/ routes are added
